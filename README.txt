PURPOSE OF PROJECT: Client/Server connection with multithreading to allow multiple users to connect to the server, and a GUI of a LogIn, SignUp, and MainMenu
VERSION or DATE: Phase 1
HOW TO START THIS PROJECT: Run WAMP, run jdbc.java to create the database "users", Run Server, Run Client(change IP address to the server's IP address)
AUTHOR: Mohamad Abou Salem
USER INSTRUCTIONS:
1. run wamp
2. open jdbc.java and run it
3. change client's ip address in the code to the server ip address (instructions found in PDF)
4. run server
5. run client
6. follow on screen commands
